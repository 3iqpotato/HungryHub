🍽️ Hungry Hub — Инструкции за инсталация
Това ръководство ще ви помогне да стартирате проекта стъпка по стъпка, дори ако никога не сте го правили преди.
---
📋 Какво ви трябва преди да започнете
Преди всичко, уверете се, че имате инсталирани следните програми:
Програма	Защо ни трябва	Линк за сваляне
Python 3.8+	Езикът, на който е написан проектът	python.org
Git	За да изтеглите кода от интернет	git-scm.com
PyCharm (по желание)	Удобна програма за работа с кода	jetbrains.com
> 💡 **Как да проверите дали Python е инсталиран?**
> Отворете Command Prompt (Windows) или Terminal (Mac/Linux) и напишете:
> ```
> python --version
> ```
> Ако видите нещо като `Python 3.11.0` — всичко е наред!
---
🚀 Стъпки за инсталация
Стъпка 1 — Изтеглете кода
Отворете терминал и изпълнете:
```bash
git clone https://github.com/yourusername/hungry-hub.git
cd hungry-hub
```
> 💡 Това изтегля проекта от интернет и ви "влиза" в папката му.
---
Стъпка 2 — Създайте виртуална среда
Виртуалната среда е като отделна "кутия" за проекта, в която се инсталират нужните библиотеки, без да засягат останалите ви програми.
На Windows:
```bash
python -m venv venv
venv\Scripts\activate
```
На macOS / Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```
> ✅ Ще знаете, че е активирана, когато пред реда в терминала се появи `(venv)`.
---
Стъпка 3 — Инсталирайте зависимостите
```bash
pip install -r requirements.txt
```
> 💡 Това инсталира всички библиотеки, от които проектът се нуждае. Може да отнеме минута-две.
---
Стъпка 4 — Приложете миграциите (настройка на базата данни)
```bash
python manage.py migrate
```
> 💡 Това създава таблиците в базата данни. Трябва да се прави само при първо стартиране или след промени в моделите.
---
Стъпка 5 — (По желание) Създайте администраторски акаунт
Ако искате да влезете в администраторския панел на адрес `/admin`:
```bash
python manage.py createsuperuser
```
Ще ви помоли да въведете потребителско име, имейл и парола.
---
Стъпка 6 — Стартирайте сървъра
```bash
python manage.py runserver
```
> ✅ Ако всичко е наред, ще видите нещо подобно:
> ```
> Starting development server at http://127.0.0.1:8000/
> Quit the server with CTRL-BREAK.
> ```
---
🌐 Достъп до приложението
Отворете браузър и отидете на:
```
http://127.0.0.1:8000/
```
Административен панел:
```
http://127.0.0.1:8000/admin/
```
---
❗ Чести проблеми
„python не се разпознава като команда"
→ Python не е добавен към PATH. При инсталацията сложете отметка на „Add Python to PATH".
„No module named django"
→ Виртуалната среда не е активирана. Върнете се към Стъпка 2 и активирайте отново.
„Port 8000 is already in use"
→ Стартирайте на друг порт:
```bash
python manage.py runserver 8080
```
