🍽️ Hungry Hub
> Платформа за онлайн поръчка и доставка на храна, свързваща потребители, ресторанти и доставчици.
![Hungry Hub](docs/screenshot.png)
---
📱 За какво служи?
Hungry Hub е уеб и мобилно приложение за доставка на храна, което обединява три типа потребители в една платформа:
Тип	Какво може да прави
🧑 Потребител	Разглежда ресторанти, поръчва храна, следи доставката
🏪 Ресторант	Управлява менюто, приема и обработва поръчки
🛵 Доставчик	Получава и изпълнява поръчки за доставка
---
✨ Основни функции
🔍 Търсене и разглеждане на ресторанти
🛒 Количка и онлайн поръчка
🚀 Бърза доставка в реално време
👤 Профили за всеки тип потребител
🔒 Сигурна автентикация с JWT токени
📱 Мобилно приложение (Android) с .NET MAUI
---
🛠️ Технологии
Слой	Технология
Backend	Django + Django REST Framework
База данни	SQLite (development)
Мобилно приложение	.NET MAUI (Android)
Автентикация	JWT (SimpleJWT)
---
🚀 Бързо стартиране
```bash
git clone https://github.com/yourusername/hungry-hub.git
cd hungry-hub
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
След това отворете браузър на http://127.0.0.1:8000/

📖 За подробни инструкции вижте docs/instructions.md
